# Gruzzolo Dev — build personale (Yahoo)

Questa è la **versione privata** dell'app, con i dati presi da Yahoo Finance e il
portafoglio reale precaricato. Si installa **a fianco** della versione pubblica
"Gruzzolo" perché ha un applicationId diverso (`it.gruzzolo.dev`) e si chiama
**Gruzzolo Dev** sul telefono.

**Non è pensata per la distribuzione**: usa gli endpoint non ufficiali di Yahoo,
adatti solo all'uso personale. Per la pubblicazione sul Play Store usa la build
"Gruzzolo" con Twelve Data.

Build: apri in Android Studio e installa, oppure `./gradlew assembleDebug`.
